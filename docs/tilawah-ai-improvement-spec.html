Design Spec · 2026-05-16

Tilawah AI Improvement

Meningkatkan akurasi evaluasi tilawah & hafalan dengan upgrade model Whisper, sequence alignment, dan deteksi mad (panjang pendek) berbasis word timestamps.

Python Sidecar Only No API Change No DB Change
Masalah

Tiga Root Cause

MASALAH 1

Model Terlalu Kecil

whisper-base (74M params) sering salah transkripsi, terutama ayat dengan harakat kompleks.

MASALAH 2

Algoritma Naif

Perbandingan posisional: satu kata salah membuat semua kata setelahnya ikut dianggap salah.

MASALAH 3

Tajweed Heuristik

Analisis mad hanya berdasarkan durasi total rekaman. Tidak akurat dan tidak bisa menunjuk kata mana yang salah.

Arsitektur

Tiga Lapis Perbaikan

Layer 1 · Transkripsi

Upgrade Model + stable-whisper

Ganti whisper-base-ar-quran (74M) → whisper-small-ar-quran (244M). Pakai library stable-ts yang mengekstrak word-level timestamps dari attention map Whisper — tanpa model tambahan.

Layer 2 · Word Matching

Sequence Alignment (difflib)

Ganti loop positional dengan difflib.SequenceMatcher. Handles insertions, deletions, reorderings — satu kata salah tidak merusak penilaian kata setelahnya.

Layer 3 · Tajweed

Deteksi Mad per Kata

Fungsi detect_mad_errors() baru: identifikasi kata dengan mad pattern dari expected text, bandingkan durasi aktual vs threshold (2/4/6 harakat). Hasil: status mad_short per kata.

Word Results

Contoh tampilan kata pada screen tilawah/hafalan

Data Flow

End-to-End Pipeline

1

Mobile → POST /evaluate

audio_base64 + expected_text + verse_number + chapter_id

2

stable-whisper transcribe

Output: transcription + word_timestamps[] (start, end per kata)

3

compare_texts (difflib)

Output: word_results[] dengan status correct / wrong / missing

4

detect_mad_errors

Update status kata yang mad-nya pendek → mad_short

5

Hitung score & feedback

Formula scoring baru, buat list tajweed feedback

6

Return response

{ transcription, word_results, word_accuracy, tajweed_score, score, feedback[] }

Scoring

Formula Penilaian Baru

tajweed_score = 100 − (jumlah mad_short × 10), min 50
completeness = 100 jika tidak ada missing, 70 jika ada

score = word_accuracy × 0.60
         + tajweed_score × 0.25
         + completeness × 0.15

Tajweed score kini berbasis data nyata (mad_short count) — bukan heuristik durasi total rekaman.
