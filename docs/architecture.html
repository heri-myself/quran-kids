AI Pipeline — Tilawah & Hafalan
Step 1
Rekam Audio
User membaca ayat, Expo Audio merekam dalam format HIGH_QUALITY. File dibaca sebagai base64 string.
expo-av
Step 2
Backend Proxy
Fastify menerima request, validasi dengan Zod, lalu forward ke Python sidecar di port 8001.
Fastify + Zod
Step 3
Transkripsi ASR
stable-whisper menjalankan Whisper large-v3-turbo dengan word_timestamps=True, menghasilkan teks Arab + timestamp per kata.
Whisper large-v3-turbo
Step 4
Komparasi Kata
difflib SequenceMatcher membandingkan teks expected vs transkripsi (setelah strip harakat). Menghasilkan status: correct / wrong / missing.
difflib
Step 5
Deteksi Mad
Kata "correct" yang mengandung pola mad dicek durasi via word timestamp. Jika < 0.35 detik → status mad_short.
Tajweed Engine
Step 6
Scoring
Skor final: 60% word accuracy + 25% tajweed score + 15% completeness. Dikirim ke mobile untuk ditampilkan.
Python
60%

Word Accuracy

Persentase kata yang benar dari total kata yang diharapkan (status "correct" atau "mad_short")

25%

Tajweed Score

Dimulai dari 100, dikurangi 10 per kata mad_short. Minimal 50. Mengukur panjang-pendek bacaan.

15%

Completeness

100 jika tidak ada kata missing (tidak dibaca), 70 jika ada kata yang dilewati.

Pola Mad yang Dideteksi

فَا Fathah + Alef
فَى Fathah + Alef Maqsurah
فَٰ Fathah + Superscript Alef (U+0670)
فِي Kasrah + Ya
فُو Dammah + Waw 
# normalize_arabic — strip harakat sebelum komparasi
def normalize_arabic(text: str) -> str:
    # includes superscript alef (U+0670) which Whisper doesn't produce
    harakat = 'ًٌٍَُِّْٰٕٓٔ'
    return ''.join(c for c in text if c not in harakat)

# contoh:
# "الرَّحْمَٰنِ" → "الرحمن"  (match dengan transkripsi Whisper)
# "الرَّحِيمِ"  → "الرحيم"

Sistem Gamifikasi

⭐ Poin Tilawah

Skor 100 → +75 poin
3 bintang → +50 poin
2 bintang → +25 poin
1 bintang → +10 poin

📖 Poin Hafalan

Avg ≥ 90 → +50 poin
Avg ≥ 70 → +30 poin
Avg < 70 → +15 poin

🔥 Streak Harian

+5 poin bonus jika baca hari berturut-turut. Streak reset jika lewat 1 hari. Dicek via lastReadAt.

🏆 Level

Level naik otomatis dari total poin. Dihitung oleh getLevelFromPoints() di backend setelah setiap sesi.

🥇 Badge

Tipe: stories_completed, streak_days, points. Diberikan otomatis saat requirement terpenuhi.

📱 State Management

Zustand stores: useProfileStore, useGamificationStore, useLastActivityStore
