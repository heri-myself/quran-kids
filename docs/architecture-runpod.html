03 — Yang Berubah vs Tidak
✏️ Yang Berubah
  • Buat runpod_handler.py — wrapper Whisper untuk RunPod Worker
  • Buat Dockerfile.runpod — Docker image GPU dengan model baked-in
  • Ganti endpoint di backend dari localhost:8001 ke RunPod API URL
  • Tambah RUNPOD_API_KEY di environment variable VPS
✅ Yang Tidak Berubah
  • Seluruh tajweed_engine.py — logic tajwid tetap jalan di VPS
  • Scoring formula accuracy×0.6 + tajweed×0.25
  • Response format ke mobile — tidak ada perubahan apapun di app
  • Kecerdasan model Whisper — sama persis, hanya lokasi eksekusi yang pindah
